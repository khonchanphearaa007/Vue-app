# Vue Jobs Project 
## Usage
This project uses JOSN-Server for a mock backend.

- You can read this document.txt learn more for install package..

# Install Dependencies
```bash
npm install
```

## Run Json Server
The server will run on port http://localhost:8000

```bash
npm run server
```

## Run vite Frontend 
Vue will run on port http://localhost:3000/

```bash
npm run dev
```

## Build for Production

```bash
npm run build
```

## Preview Production Build
```bash
npm run preview
```
